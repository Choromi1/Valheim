Valheim Dedicated Server

발헤임 모드 적용 가이드

이 가이드는 우리 서버 접속을 위한 일반 유저용 안내입니다. 모드는 하나씩 설치하지 않고, 공유받은 r2modman 프로필 코드로 가져옵니다.

적용 전 준비 및 프로필 Import

아래 순서대로 진행하면 모드를 한 번에 적용할 수 있습니다.

프로필 코드 입력 영역

여기에 r2modman 프로필 코드를 입력하세요

GitHub에 올리기 전 index.html에서 위 문구를 실제 코드로 바꾸면 됩니다.

진행 순서

1

Valheim 1회 실행

Steam에서 발헤임을 한 번 실행해서 기본 폴더를 생성합니다.

2

r2modman 설치

Thunderstore에서 r2modman을 설치하고 실행합니다.

r2modman 페이지 열기
3

Valheim 선택

Valheim Dedicated Server가 아니라 Valheim을 선택합니다.

4

Import / Update

프로필 화면에서 Import / Update를 클릭합니다.

5

From code 선택

Import new profile → From code를 선택하고, 위 프로필 코드를 붙여넣습니다.

6

Start modded 실행

모드 설치가 끝나면 Start modded로 실행합니다. Steam 일반 실행은 사용하지 않습니다.

게임 실행 후 문제가 없다면 정상 적용 완료!

비주얼 모드 설치 방법 · Nexus Mods

셰이더/텍스처 계열은 프로필 Import에 포함되지 않을 수 있어 별도 설치가 필요합니다.

NEXUSMODS https://www.nexusmods.com/valheim
1Nexus Mods 접속
2비주얼 모드 다운로드
3압축 해제
4안내 경로에 복사
5Start modded 실행

주의사항

  • 모든 유저는 반드시 같은 프로필을 사용해야 합니다.
  • 모드를 추가/삭제하거나 버전을 변경하면 접속이 불가능할 수 있습니다.
  • 비주얼 모드는 유저마다 다르게 적용 가능하며 서버 영향이 없습니다.
  • 문제가 생기면 모드를 삭제하지 말고 먼저 문의해주세요.

적용 모드 상세 설명

프로필에 포함되는 모드와 역할입니다.

필수 & 코어

BepInExPack_Valheim모드 실행에 필요한 핵심 프레임워크입니다.
Jotunn여러 모드 개발에 필요한 라이브러리입니다.
YamlDotNetYAML 설정 파일을 읽기 위한 라이브러리입니다.

Azumatt 편의 모드

AzuCraftyBoxes상자 재료를 바로 제작에 활용할 수 있습니다.
AzuAutoStore주변 상자에 아이템을 자동 정리합니다.
AzuExtendedPlayerInventory인벤토리와 장비 슬롯 편의성을 확장합니다.
SleepSkip다른 유저가 잠든 상태를 반영해 밤을 넘길 수 있습니다.
Where_You_At지도에 플레이어 위치를 공유합니다.
Recycle_N_Reclaim아이템/구조물을 회수하거나 재활용합니다.
AzuAreaRepair망치 수리 시 주변 건축물을 함께 수리합니다.

기능 확장

PlantEverything거의 모든 곳에 작물을 심을 수 있게 해줍니다.
TeleportEverything아이템/생물/플레이어 텔레포트 기능을 확장합니다.
FastTeleport포탈/던전 이동 시간을 단축하고 로딩을 개선합니다.
FloorsAreRoofs바닥 타일을 지붕처럼 인식해 건축 편의성을 높입니다.

전투 & 생존

Custom_Raids다양한 커스텀 레이드 이벤트를 추가합니다.
Deathlink사망 관련 기능을 조정합니다.
SkilledCarryWeight스킬 레벨에 따라 최대 소지 무게가 증가합니다.

인벤토리 & UI

ItemStacksItemWeights아이템 스택 수와 무게 정보를 조정합니다.
Quick_Stack_Store_Sort_Trash_Restock빠른 저장/정리/버리기/재보충 기능을 제공합니다.
MyLittleUIUI를 개선하고 유용한 정보를 표시합니다.
EmoteWheelReworked감정표현 휠 메뉴를 더 편하게 사용할 수 있습니다.

지도 & 탐험

BetterCartographyTable지도 제작대 기능을 확장하고 다양한 설정을 제공합니다.

알아두세요!

  • 모든 유저는 서버 클라이언트가 동일한 모드와 버전을 사용해야 정상 작동합니다.
  • 모드를 임의로 추가/삭제하거나 버전을 변경하면 접속이 불가능할 수 있습니다.
  • 새로운 모드가 추가되면 서버에서 새로운 프로필 코드를 제공할 예정입니다.
