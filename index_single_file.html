Install Flow

기능 모드 설치 방법 · r2modman

기능 모드는 개별 검색 설치가 아니라, 프로필 코드를 가져오는 방식으로 맞춥니다.

r2modman 다운로드
1

Valheim 1회 실행

Steam에서 발헤임을 한 번 실행해 기본 폴더를 생성합니다.

2

r2modman 설치

Thunderstore에서 r2modman을 설치하고 실행합니다.

3

Valheim 선택

Valheim Dedicated Server가 아닌 Valheim을 선택합니다.

4

Import / Update

프로필 화면에서 Import / Update를 클릭합니다.

5

From code 선택

Import new profile에서 From code를 선택하고 공유받은 프로필 코드를 붙여넣습니다.

6

Start modded

모드 설치가 끝나면 Steam이 아닌 Start modded로 실행합니다.

Troubleshooting

문제가 발생했다면?

1. Start modded 확인Steam 일반 실행이나 Start vanilla로 실행하면 모드가 적용되지 않습니다.
2. 최신 프로필 Import모드가 추가되면 이전 프로필 대신 최신 코드로 다시 Import하세요.
3. 화면 캡처 전달Mods 탭, 오류 메시지, 막힌 단계 화면을 캡처해서 넬에게 보내주세요.

적용에 어려움이 있다면 넬에게 문의하세요!
